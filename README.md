#ResponsiveWeb
##项目介绍
一个简单的**响应式**页面（不支持ie8及以下）
##所用插件
1.  OwlCarousel2:实现广告轮播
2.  Picturefill:解决ie9对<&lt;picture&gt;元素不支持的问题
